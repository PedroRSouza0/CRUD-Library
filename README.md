#  CRUD Library

Sistema de gerenciamento de biblioteca desenvolvido em Python com integração em um banco de dados local SQLite. O projeto foi criado com o intuito de exercitar lógica de programação juntamente com a integração dos dados em um Banco de Dados, podendo pratica os comandos SQL.

O Projeto que está em desenvolvimento, foi até o momento feito utilizando apenas programação estruturada e o uso da biblioteca sqlite3 integrada ao python.

##  Requisitos Funcionais

-  Cadastro de livros e Clientes  
-  Consulta de registros  
-  Atualização de dados  
-  Remoção de livros e Clientes

### Melhorias Futuras
- Código orientado a objeto e com tratamento de erros e exceções
- Interface gráfica
